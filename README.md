# Engineering Data Analysis Tool (Python)

## 📊 Overview
A specialized data processing tool designed to analyze engineering signals and perform statistical smoothing. This project showcases the ability to handle numerical datasets and generate professional-grade visualizations.

## 🛠️ Technical Highlights
* **NumPy Integration:** Utilized for high-performance numerical simulation and signal generation.
* **Pandas Workflow:** Implemented Rolling Window Calculations (SMA) and boolean indexing for anomaly detection.
* **Matplotlib Visualization:** Developed multi-layer plots to differentiate between raw data, processed trends, and specific data peaks.

## 💻 Tech Stack
* **Language:** Python
* **Libraries:** NumPy, Pandas, Matplotlib

## ⚙️ How to Run
1. Install dependencies:
   `pip install pandas numpy matplotlib`
2. Execute the script:
   `python analysis.py`
